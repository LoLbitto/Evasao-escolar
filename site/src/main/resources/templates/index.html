ESTUDANTES CORPO DOCENTE SOLICITAR INFORMAÇÔES

Projeto que tem como objetivo reduzir as taxas de evasão escolar utilizando Java e Fuzzy Logic(lógica nebulosa) para identificar os estudantes mais propensos a evadirem a vida escolar. A entrada de informações referentes a um estudante pode ser feita por uma instituição de ensino, pela família ou pelo estudante. Demais pessoas — além das mencionadas — não devem fazer o envio, pois são necessárias informações pessoais como nome completo, número de matrícula e renda familiar.

Projeto que visa previnir a evasão escolar.
